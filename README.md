# 🎟️ TicketBlaster

TicketBlaster is a full-stack web application inspired by modern ticketing platforms like Ticketmaster.  
The goal of the project is to explore how event discovery and ticket access can be made more transparent, user-friendly, and fair — while building a real-world PHP + MySQL application from scratch.

This project was developed as part of my Web Development course and serves as a portfolio piece demonstrating backend development, database integration, and user authentication.

---

## 🚀 Features

- User sign-up and login with secure password hashing
- Dynamic event listings powered by a MySQL database
- User dashboard with personalized content
- Server-side rendering using PHP
- Modular PHP structure using includes
- Clean separation between frontend and backend logic

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** TopHat
- **Version Control:** Git & GitHub

## 🌐 Deployment

TicketBlaster is currently hosted on a school-managed server (TopHat) used for academic coursework.  
Because this environment is not intended for permanent public hosting, this repository serves as the primary reference for the project’s architecture, functionality, and code quality.

A public deployment for portfolio purposes may be added in the future.
